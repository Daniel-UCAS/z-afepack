AFEPack是一套为自适应有限元编程设计的C++类库.

此包的强大性在于，它可以使您在一两小时内完成复杂的偏微分方程组的有限元编程工作；而且使您的程序天生会具有相当大的可重用性：解一个新的问题常常只需要在旧程序上进行少许修改就能完成，而很多并不是太高要求的问题都可以在库中附带的例子上进行不多的修改即可解决。

希望做贡献的人不妨考虑：
 * 完善文档
 * 添加更多模板单元
 * 应用到适当的实际算例
